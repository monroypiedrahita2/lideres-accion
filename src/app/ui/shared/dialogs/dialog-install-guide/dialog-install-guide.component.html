install_mobile Instalar Aplicación

Sigue estos pasos para instalar la aplicación en tu dispositivo y acceder más rápido.

ios_share iPhone / iPad

  1. Abre esta página en Safari O Chrome.
  2. Toca el botón Compartir o añadir a inicio ios_share en la barra inferior.
  3. Desliza y selecciona "Agregar al inicio" (Add to Home Screen).
  4. Confirma tocando "Agregar".

android Android / Chrome

  1. Abre el menú del navegador (tres puntos ⋮).
  2. Selecciona "Instalar aplicación" o "Agregar a la pantalla principal".
  3. Confirma la instalación.

computer Escritorio / Chrome

  1. Busca el ícono de instalación install_desktop en la barra de direcciones.
  2. O abre el menú del navegador (tres puntos ⋮) y selecciona "Instalar aplicación".
  3. Confirma la instalación.
